Pattern 1 — The Segmentation Clock

Wnt / Notch oscillator

Number of contiguous Pro (P) codons after ATG start determines the number of body segments. Real basis: Wnt and Notch signaling creates a molecular clock in developing embryos that pulses once per segment boundary.

Pattern 2 — Bilateral Symmetry Lock

Pitx2 / Nodal laterality axis

Every Val (V / GTT) codon produces a bilateral (left+right) pair — never a single appendage. Real basis: Pitx2 and Nodal signaling establish left-right asymmetry; Distal-less (Dll) genes then specify the bilateral appendage bud.

Pattern 3 — HOX Positional Identity

HOX homeodomain gene cluster

Position in sequence → anatomical region. The first morphemes after ATG are applied to the HEAD region. Middle morphemes build the TORSO. Last morphemes before STOP become TAIL/POSTERIOR structures. This directly mirrors the real HOX gene anterior-posterior expression gradient.

Pattern 4 — Appendage Identity Rule

Distal-less (Dll) appendage specifier

What codon follows Val determines what body part grows on that branch. The branch (Val) activates Dll; the next codon's morpheme specifies what Dll builds. This is exactly how selector genes work in real appendage specification.

Pattern 5 — Pigmentation Cascade

MC1R / TYRP1 / ASIP melanin synthesis

The sequence of Tyr (Y/TAT) and Thr (T/ACT) codons determines pigment pattern output. Tyr is the literal melanin precursor (L-DOPA pathway); Thr is a stripe-band activator. This mirrors real MC1R-driven melanin switching.

Pattern 6 — Androgen Amplifier

RXFP2 / testosterone receptor axis

Arg (R/CGT) codons in the first 30% of the sequence amplify ALL subsequent morphemes. This models the real RXFP2 androgen receptor found in Bighorn Sheep and Soay Sheep — where it drives not just horn size but overall morphological intensity. The more Arg early in sequence, the more extreme the phenotype.
